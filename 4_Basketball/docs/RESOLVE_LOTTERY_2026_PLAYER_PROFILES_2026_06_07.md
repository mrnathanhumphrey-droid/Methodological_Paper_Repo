# Methodology footnote

Every number in the profile tables comes from `data/parquet/resolve_lottery_2026.parquet`. The Resolve Score formula is deterministic:

```
resolve_score = (
      0.45 * v3_rank_pct         # outcome-calibrated GBM
    + 0.20 * hand_rank_pct       # production composite
    + 0.30 * advanced_signal_pct # PSI + STD + FT rate + def event rate
    + survivorship_penalty       # -0.03 * max(0, signal-5)
    + intl_low_n_penalty         # -0.12 for intl with league_n<10 + no NCAA PBP
)
```

Same inputs → same output. Consensus boards were pulled 2026-06-07 AFTER our board was locked, used for orientation only, not as model inputs. Cardinal rule held: "we don't project on other's projections ever. raw data only."
