# Pattern 033 — Civil-war-mass-displacement as candidate Type E (5th conflict-type)

- **ID:** PATTERN_033
- **Status:** candidate-hypothesis (4-6 case cluster, emergent from Paper 4 P4-C classifier fire)
- **Type:** mechanism (NEW candidate conflict-type)
- **Discovered:** 2026-05-27 via PRE_REG_018 first fit (unpredicted)
- **Severity / interest:** HIGH — emerged unpredicted; could be 5th type

## One line

**Type E civil-war-mass-displacement**: state-vs-rebel conflicts where civilian populations flee at extreme rates, producing displacement-per-fatality ratios in the 300-2300 range. Distinct from Type A formal-army (DPF ≤ 150) and Type B predator-militia (one-sided dominant). Anchor candidates: **SYR (2300 DPF), YEM (615), AFG (349), IRQ (1473), LBY (1111), CMR (1486), CAF (938)**.

**Mechanism signature**:
- State-share ≥ 60% (state-dominated like Type A) BUT
- DPF ≥ 300 (much higher than Type A) AND
- Admin-1 dominance variable (30-97% — heterogeneous)
- Multi-actor non-state landscape often present (low-to-moderate strife share, with one-sided present)
- Mass civilian displacement is PRIMARY OUTCOME

**Distinct from Type A formal-army** (ETH/UKR/RUS/ISR/PAK/EGY at DPF 30-63): Type E has state-vs-rebel conflict with prolonged civilian-displacement outcome, while Type A is short-duration high-intensity state-vs-state with low DPF.

**Distinct from Type B predator-militia** (COD/MOZ/HTI at DPF 674-1269): Type B is one-sided-dominated; Type E is state-based-dominated with mass-displacement.

**Distinct from Type C irregular insurgency** (MEX/BRA/SDN/COL): Type C is mixed; Type E is more state-dominated.

## Why it stands out

**Emerged UNPREDICTED in PRE_REG_018 fire**. The 3-type framework (A/B/C) didn't anticipate this cluster. 4-6 cases with consistent signature suggests a real structural class.

**Mechanism candidate**: Type E represents **prolonged civil wars in low-state-capacity contexts**. The displacement-per-fatality ratio is high because:
- Civilians flee proactively (war anticipation) not reactively (single events)
- Multiple displacement events per person inflate count
- Cross-border refugee flows often interact with internal displacement
- Long duration (5+ years for most cases) accumulates displacement

**Examples by mechanism**:
- SYR (2300 DPF): 12-year civil war, multi-faction (Assad + rebels + Kurds + ISIS + Turkey + US)
- YEM (615 DPF): Houthi vs Hadi government + Saudi coalition + UAE-backed STC
- AFG (349 DPF): Taliban return 2021 + ISKP attacks ongoing
- IRQ (1473 DPF): Post-ISIS reconstruction + ongoing skirmishes
- LBY (1111 DPF): GNU vs LNA + tribal militias
- CMR (1486 DPF): Anglophone separatist crisis + Boko Haram spillover
- CAF (938 DPF): CPC militia coalition vs FACA + Russian mercenaries
