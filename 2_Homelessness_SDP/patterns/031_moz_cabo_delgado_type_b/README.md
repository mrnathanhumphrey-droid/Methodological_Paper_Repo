## Why it stands out

**Second Type B predator-militia confirmation beyond COD.** Mechanism replication:
- Single-actor dominance (ASWJ 82.5% vs COD's ISCAP 54%)
- Hyper-concentrated geographically (97% in Cabo Delgado vs COD's 83% in NK+Ituri)
- Civilian-targeting (one-sided > state-based)
- Mass-displacement outcome (3.6M displaced / 5K fatalities = 715 ratio, similar to COD's 783)

**Cross-link to Paper 2 disaster regimes**: MOZ classifies Regime 3 storm-dominant (Cyclone Idai 2019, Kenneth 2019). MOZ is **bi-regime country**: Type B conflict + Regime 3 storm simultaneously. Twin-paper interface candidate.
