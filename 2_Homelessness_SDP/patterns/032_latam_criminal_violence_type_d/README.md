## One line

**Type D criminal-violence**: cartel/gang violence dominated by non-state-vs-non-state combat, with LOW displacement-per-fatality (≤100) — distinct from Type B (predator-militia, 250-800 DPF) and Type C (irregular insurgency, 100-300 DPF). Anchor cases: **MEX 99.5% non-state / 30 DPF, BRA 95.2% non-state / 6 DPF, ECU 99.7% non-state / 38 DPF.**

## Why it stands out

**Resolves PRE_REG_018 H2** (4th-type question). Cartel/gang violence is structurally distinct from:
- Type B predator-militia: Type B targets civilians for displacement (DPF 250-800); Type D targets rival cartels + selected civilians (DPF <50)
- Type C irregular insurgency: Type C has political-ideological motivation + multi-channel state-vs-rebel landscape; Type D is purely commercial (drug markets, territory)
- Type A formal-army: Type A is state-vs-state; Type D is non-state-vs-non-state

**Mechanism**: Organized criminal violence in Latin America (MEX cartels, BRA PCC vs CV vs militia, ECU post-2022 cartel war) generates high fatality rates concentrated in cartel-territory states but does NOT produce mass internal displacement at the rates of insurgency or war. Criminal violence has narrow targeting (rival cartel members + selected civilians + journalists) rather than broad civilian displacement.

**Cross-link to Paper 1 executive-aggrandizement**: MEX is also a Federal-friction slow-burn libdem case (PATTERN_026 P1-B; 3/3 conditions in PRE_REG_010). **MEX is bi-mechanism**: Type D criminal-violence + federal-friction backsliding. Cross-paper case.
